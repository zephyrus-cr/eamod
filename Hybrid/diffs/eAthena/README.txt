ALL FILES IN THIS FOLDER ARE DESIGNED TO BE OPTIONAL PATCHES

THEY ARE MADE SPECIFICALLY FOR EATHENA EAMOD

INSTALLATION INSTRUCTIONS
================================

Patch any of the files in this folder to your eA/ folder.


DESCRIPTION OF FILES
================================


instance.patch	-	v 1.0
================================
Patch this in to your eAmod to enable the creation of
instances that can be attached to players and guilds instead
of just parties.

reload.patch	-	v 1.0
================================
Adds in @reloadfactiondb and @reloadlangdb to allow for
setting changes without the need to restart the server
